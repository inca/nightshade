Nightshade

Extensible base for application CSS frameworks

GitHub

What's inside?

  • minimal conventions for creating CSS specs for your team
  • 4KB min/gzip CSS core containing only stuff you'll end up implementing anyway (buttons, inputs, grids and few helpers) — and nothing more
  • easily extensible via Stylus (not intended for download-and-use)

Please refer to README on GitHub for conventions and usage.
